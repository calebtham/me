How to use:
- Drag and drop the Source node (Green square)
- Drag and drop the Destination node (Red square)
- Drag across the grid to draw walls (Black squares)
- Press ENTER to start the search, and ENTER again to reset the grid
- Press 'a' to toggle between Dijkstra's and A* search
- Press ESC to clear the grid

Key:
Green = Source; Red = Destination; Black = Wall; Light-Blue = Explored; Dark-Blue = Frontier;

Mode: A* Search
